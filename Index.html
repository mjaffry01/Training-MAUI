Capstone Project

Duration: 5 Hours

Project Overview:

Participants will apply all the concepts learned throughout the training to build a complete, real-world .NET MAUI application. This project will encompass UI design, data handling, platform-specific features, performance optimization, security, testing, and deployment.

Project Phases:

Phase 1: Planning and Design

Duration: 1 Hour

  • Activities:
    • Define the application scope and requirements.
    • Design UI/UX mockups using tools like Figma or Sketch.
    • Plan the application architecture using MVVM.
  • Deliverables:
    • Project Requirements Document
    • UI/UX Mockups
    • Architectural Diagram
Phase 2: Development

Duration: 3 Hours

  • Activities:
    • Implement UI using XAML and custom controls.
    • Integrate data services and consume APIs.
    • Add platform-specific features as needed (e.g., camera access).
    • Ensure performance optimizations and implement security measures.
  • Deliverables:
    • Functional .NET MAUI Application
    • Source Code Repository (e.g., GitHub)
Phase 3: Testing and Deployment

Duration: 1 Hour

  • Activities:
    • Write and execute unit and UI tests.
    • Optimize performance based on profiling results.
    • Prepare the app for deployment to selected platforms.
    • Deploy the application to app stores or as a local installer.
  • Deliverables:
    • Test Reports
    • Deployed Application
    • Deployment Documentation

Suggested Projects:

  • To-Do List Application: Manage tasks with CRUD operations, data persistence, and UI enhancements.
  • Weather Forecast App: Fetch and display weather data from a public API with responsive UI.
  • Simple E-Commerce App: Manage products, cart, and checkout processes with basic authentication.

Presentation and Evaluation:

  • Activity: Present the completed project, highlighting key features and learning outcomes.
  • Evaluation: Peer reviews and instructor feedback based on predefined criteria (e.g., functionality, design, code quality).
